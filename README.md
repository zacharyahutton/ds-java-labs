# DS Java Labs

> **Portfolio demonstration** — UTech Data Structures coursework labs in Java. Representative implementations for learning and portfolio review, not proprietary exam submissions.

## Problem

Implement and analyse core data structure operations: binary search trees, traversals, and complexity analysis.

## Stack

- Java 17+ · JUnit 5 · Maven

## Labs included (starter)

| Lab | Topics |
|-----|--------|
| `BinarySearchTree` | insert, search, in-order traversal |
| `ComplexityNotes` | Big-O write-ups for tree operations |

## Quick start

```bash
mvn test
```
