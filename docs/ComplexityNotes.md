# Binary search tree (average case)

| Operation | Time | Notes |
|-----------|------|-------|
| insert | O(h) | h = tree height; O(log n) when balanced |
| search | O(h) | Same as insert path |
| in-order traversal | O(n) | Visits every node once |

# Linked list

| Operation | Time |
|-----------|------|
| addFirst | O(1) |
| contains | O(n) |

# Chained hash map

| Operation | Average | Worst |
|-----------|---------|-------|
| put / get | O(1) | O(n) with many collisions |
